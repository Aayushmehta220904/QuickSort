## QuickSort

## Organize your files in one click

QuickSort is a lightweight desktop utility that automatically organizes files into folders based on their file type. It is designed for everyday users who want a clean Desktop or Downloads folder without complex settings, accounts, or internet access.

## Official website & downloads:
https://quicksortfiles.vercel.app

## Why QuickSort?

Modern desktops get messy fast screenshots, PDFs, installers, images, code files, all mixed together. QuickSort solves this with a simple, safe, one click workflow.

No configuration
No learning curve
No background processes
No internet required

Just select a folder and organize.

## Features

One-click file organization  
Automatically sorts files by extension/type  
Creates clean, readable folders  
Works completely offline  
Never deletes your files  
Simple, minimal interface  
Available for **macOS** and **Windows**

## How it works
1. Open **QuickSort**
2. Select a folder (Desktop, Downloads, or any directory)
3. Click **Organize**

QuickSort moves files into neatly grouped folders such as Documents, Images, Videos, Code, Archives, and more.
