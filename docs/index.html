QuickSort

Organize your files in one click.

Download for macOS Download for Windows

What does QuickSort do?

  • Automatically organizes files by type
  • Documents, images, videos, code files & more
  • Works completely offline
  • Does not delete or modify your files

How to use

  1. Open QuickSort
  2. Select a folder (Downloads / Desktop)
  3. Click Organize

Security notes

macOS: Right-click the app → Open → Open (unsigned developer warning)

Windows: Click “More info” → “Run anyway” (SmartScreen)
